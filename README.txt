==================================
## Required
- Agent version 20.0+
- Java 8


## Deployment steps
- Copy SFFCUAgentPlugin.jar file under <agent-install-dir>/ver.x.x.x.x/sdk-plugins
- Add this system property java option in startup

    -Dallow.unsigned.sdk.extension.jars=true
    
- Restart the JVM
